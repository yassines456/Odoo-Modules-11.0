Instagram, Pinterest and Dribbble in website footer

This module allow to display Instagram, Pinterest and Dribbble on the website. Links will be presented the same way as other social links in Odoo (icons).

Add link to your social media

Simply fill in the fields in the Settings / Companies. No programming or technical knowledge required.

Icons are displayed

Same way as the original icons - when visitor clicks on the icon the social media website will be displayed.
